# SEO, conversión y enlaces

## Objetivo de conversión

Llevar al visitante a comprar mediante Pagopar o consultar por WhatsApp.

## Pagopar

- Usar enlaces externos.
- No integrar API.
- Abrir en nueva pestaña cuando corresponda.
- Añadir etiquetas de analítica a cada CTA.
- Centralizar enlaces por plan.

```ts
export const paymentLinks = {
  fullPrelaunch: "",
  fullLaunch: "",
  installmentsPrelaunch: "",
  installmentsLaunch: "",
};
```

## WhatsApp

- Admitir varias asesoras.
- Cada asesora puede tener nombre, número y mensaje propio.
- Usar `wa.me`.
- Mensaje ejemplo: “Hola, vi la landing de La Trilogía TOL y quiero recibir más información sobre la inscripción.”

## Requisitos SEO

- title y description;
- canonical;
- Open Graph;
- Twitter Card;
- schema.org cuando corresponda;
- headings coherentes;
- URLs limpias;
- alt descriptivos;
- imágenes optimizadas;
- sitemap y robots;
- contenido visible en HTML;
- buen rendimiento móvil.

## Analítica

Dejar preparados eventos para clic en Pagopar, clic en WhatsApp, reproducción de video, apertura de FAQ, scroll profundo y CTA final.

No insertar IDs reales de Analytics o Pixel hasta recibirlos.
