# Sistema visual

## Paleta oficial Academia GPS

- Azul principal: `#004877`
- Turquesa: `#008080`
- Verde: `#76A858`
- Crema: `#F8F2EA`
- Amarillo: `#E6C619`

## Uso recomendado

- Azul: títulos, fondos institucionales, navegación, textos importantes.
- Turquesa: botones secundarios, acentos, TOL-Mark.
- Verde: indicadores positivos, beneficios, pequeños acentos.
- Crema: fondos suaves y secciones de descanso visual.
- Amarillo / dorado: urgencia, bonos, TOL-Coach, llamados destacados.
- Blanco: fondos limpios y contraste.
- Evitar usar todos los colores con la misma intensidad en una misma sección.

## Tipografías

- Títulos: Poppins.
- Texto y navegación: Montserrat.
- Usar fuentes web optimizadas.
- No usar más de dos familias tipográficas.

## Dirección visual

La web debe sentirse premium, profesional, humana, confiable, ejecutiva, relacionada con salud y gestión, y clara para profesionales no técnicos.

## Componentes visuales

- encabezado simple;
- cintillo superior de urgencia;
- hero con propuesta fuerte;
- tarjetas para TOL-FI, TOL-Mark y TOL-Coach;
- bloques de beneficios;
- cronología o pasos de metodología GPS;
- módulos de oferta;
- garantía;
- acordeón de preguntas frecuentes;
- CTA final;
- footer institucional.

## Fotografías

Usar las fotografías profesionales entregadas de la Dra. Nidia.

Preferencias:

- hero: foto con espacio negativo para texto;
- autoridad: retrato sentado o de pie;
- recortar sin deformar;
- optimizar a WebP/AVIF;
- conservar tono natural;
- no aplicar filtros intensos.
