# Trilogía TOL — paquete de contexto para Claude / Codex

Este paquete evita volver a explicar el proyecto desde cero y reduce el consumo de tokens.

## Orden de lectura recomendado

1. `PROJECT_CONTEXT.md`
2. `DECISIONS.md`
3. `STATUS.md`
4. Leer solo el archivo específico necesario:
   - diseño: `DESIGN_SYSTEM.md`
   - estructura y contenido: `CONTENT_MAP.md`
   - desarrollo: `TECHNICAL_SPEC.md`
   - SEO, conversión y enlaces: `SEO_CONVERSION.md`
5. Usar `AI_MASTER_PROMPT.md` como prompt inicial.

## Regla principal

No reconstruir el proyecto desde cero si ya existe código. Primero inspeccionar el repositorio, identificar lo ya implementado y continuar de forma incremental.
